# 輸入元件

通用的 Compose 輸入元件，集中於 `util/TextInputField.kt` 與 `widget/QKeyboard.kt`。

## InputStateWrapper

`util/TextInputField.kt:79-82`：

```kotlin
data class InputStateWrapper(
    var state: TextFieldState = TextFieldState(),       // Compose Foundation Text2 的新 state
    var errorId: Int? = null                            // 錯誤訊息 string res id
)
```

整個 App 的輸入欄位都用這個包裝。`TextFieldState` 是 Compose Foundation `text2` 套件的新型別（實驗 API），透過 `textAsFlow()` 提供 reactive 訂閱。

## TextInputField（`util/TextInputField.kt`）

`@Composable fun TextInputField(textFieldValue: InputStateWrapper, placeholder, keyboardType, ...)` — 包裝 `BasicTextField2`。

### 功能特性

- 自動 focus border 顏色變化（focus 時 2dp ColorBlue, 失焦 0dp）
- 內建 clearable trailing icon（X 按鈕清空）
- 支援 maxLength（用 `AndroidInputTransformation` 自動截斷）
- 自動 bringIntoView（鍵盤彈出時捲到可見）
- 鍵盤類型支援 Number / Text / Search / Done / Next 等

### maxLength 處理

`AndroidInputTransformation`（line 238-247）：

```kotlin
override fun transformInput(originalValue, valueWithChanges) {
    if (valueWithChanges.length > maxLength) {
        valueWithChanges.delete(maxLength, valueWithChanges.length)
    }
}
```

## QKeyboard（`widget/QKeyboard.kt`）

自製數字鍵盤，給 [[SetStockPage]] 庫存數量輸入用（**但實際上 SetStockPage 把這段註解掉了**，目前沒實際使用）。

### KeyBoardEvent sealed class

```kotlin
sealed class KeyBoardEvent {
    data class AddValue(value)
    object DeleteValue
    object ClearValue
    object OnDismiss
}
```

### 配置

3x4 鍵：1/2/3 ◇ 4/5/6 ◇ 7/8/9 ◇ C/0/Del 或 C/0/OK（依是否傳 `onClickOk`）

選項：
- `maxLength` — 限制長度
- `isNum` — 開啟時自動去掉前導 0
- `fontSize` — 字級
