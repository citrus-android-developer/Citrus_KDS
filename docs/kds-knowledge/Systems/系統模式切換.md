# 系統模式切換

App 支援兩種模式，由 `prefs.mode` 決定（`0=KDS`, `1=OrderReady`）。模式儲存在 SharedPreferences，App 重啟保留。

## 切換流程

```
SettingPage 的 ModeRadio (SettingPage.kt:520-565)
        │ onClick
        ▼
event(OnModeChanged(it))                ← CentralViewModel.kt:285
  prefs.mode = event.mode
  state.modeState = event.mode
        │
        │ 同時 navigateTo("kds")        ← SettingPage.kt:355
        ▼
KdsScreen 進入後
LaunchedEffect(modeState) {              ← KdsScreen.kt:90-94
  if (modeState == 1) navigateToOrderReady()
}
        ▼
OrderReadyScreen 顯示
```

## 切換本身沒有 API 呼叫

`OnModeChanged` 只更新 prefs 與 state，**不打任何 API**。

進入 OrderReady 後才開始輪詢 `/controller/OrdersList`（見 [[OrderReady模式]]）。

## startDestination

`MainActivity.kt:154`：
```kotlin
startDestination = if (prefs.mode == 0) "kds" else "orderReady"
```

冷啟動依 prefs 決定。設定中切換模式時不會冷啟動，靠 `KdsScreen` 的 `LaunchedEffect(modeState)` 補路由。
